# QuikLearn - Gamified Learning SaaS

**A premium educational platform by Sudarshan Labs**  
For Class 6-12 students & competitive exam aspirants (UPSC, JEE, NEET, Olympiads)

![QuikLearn](https://via.placeholder.com/800x400/6366f1/ffffff?text=QuikLearn+Hero)

## ✨ Features
- Fully gamified experience (XP, Streaks, Badges, Leaderboard)
- Beautiful Dark/Light mode with glassmorphism
- Working Olympiad MCQ quiz engine with timer
- Mobile-first, premium SaaS UI (Notion + Duolingo vibe)
- Pure Vanilla HTML5 + CSS3 + JavaScript (no frameworks yet)
- Easy to host on Vercel / GitHub Pages / Netlify

## 🛠 Tech Stack
- **Frontend**: Semantic HTML5, Advanced CSS3 (CSS Variables, Flex, Grid, Glassmorphism), Vanilla JS (ES6+)
- **Icons**: FontAwesome 6
- **Fonts**: Google Inter
- **Future**: Firebase / Supabase backend + Authentication

## 🚀 How to Run (Local)
1. Folder download karo ya clone karo
2. `index.html` ko browser me open karo
3. Done! Koi server ki zarurat nahi
